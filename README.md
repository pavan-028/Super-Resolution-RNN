# Super-Resolution-RNN

Model description:

• Activation function used: LeakyReLu

• Normalization : Batch Normalization

• Stride : 2 and kernel: 3×3

• Total params: 9,602,898

• Trainable params: 9,600,338

• Non-trainable params: 2,560
